  • Learning 2: Initial = Every CSS property has an initial or default value. If you assign the value initial to that property then it effectivelyresets to its default value.
  • Learning 3: unset = Combination of Inherit and initial. when applied to an iherited property it set the value to iherit,to non inherited propety to initial.
  • Learning 4: revert = helps when working with new css features and you not know oof the top of your head what the default values are.
